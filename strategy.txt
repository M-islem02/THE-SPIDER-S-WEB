Spider's Web
============

Start with 13 nodes and the 2 sample PDF edges.

Files:
- `addEdge.js`: adds an edge
- `removeEdge.js`: removes an edge by id
- `updateWeight.js`: changes edge weight
- `detectAnomaly.js`: finds `weight <= 0`
- `computeMST.js`: runs Kruskal with `weight > 0`
- `app.js`: UI, scenarios, rendering, and events

Rule:
- cursed edge: `weight <= 0`
- MST edge: only `weight > 0`

After each action, the graph, MST, total weight, and anomaly report refresh.
