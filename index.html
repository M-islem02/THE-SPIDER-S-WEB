Instructions

Challenge

Build a dynamic graph with 13 nodes. Each node is a Phantom Troupe member, each edge is a Nen connection with a weight from 1 to 20.

Rules

  • Add Edge: connect two nodes with a weight
  • Remove Edge: delete a connection by ID
  • Search Node: see all connections of a node
  • MST (Kruskal): find the minimum spanning tree
  • Cursed Edge: any edge with weight ≤ 0 is cursed and excluded from MST

How to use

  1. Enter From, To, and Weight, then click Add
  2. Enter an Edge ID, then click Remove
  3. Enter a Node number, then click Search
  4. The MST updates automatically after every action
  5. Cursed edges appear in RED and are excluded
